# Capstone Project Team 4

## 🚀 Quick Start

### Prerequisites
- Python 3.9+
- Git

### Installation
```bash
git clone https://github.com/COSC-499-W2025/capstone-project-team-4.git
cd capstone-project-team-4
pip install -r requirements.txt
```

### Usage
```bash
# Run the main CLI
python -m src.main --help

# Run commands
python -m src.main [OPTIONS] COMMAND [ARGS]...      
```

### Testing
```bash
pytest
pytest --cov=src --cov-report=html
```
