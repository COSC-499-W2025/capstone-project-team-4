# Team 4 Term 2 Week 10

## Milestone recapped

This week the team continued developing major system features across both the frontend and backend. Work included implementing project data fetching and analysis features, expanding the analysis UI, adding snapshot comparison functionality, and introducing a portfolio frontend skeleton. The ResumeBuilder was also extended with new form components, export utilities, and a page with live preview. Alongside these features, the team addressed smaller bug fixes, refactored parts of the backend, and made improvements to testing and validation to ensure the system remains stable.
